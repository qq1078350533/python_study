1：隐式转化优先级：byte，short,char -- int -- long -- float -- double（最高）
2:++参与其他操作的时候:
            ++在后面，先拿变量做操作，然后变量在++
            ++在前面，先变量++，然后再拿变量做操作
3：逻辑运算符：&:有假全为假，全真才为真
            |:有真全为真，全假才为假
            ^:相同为假，不同为真
            !:真则假，假则真
4: 三元运算符：
          关系表达式？表达式1：表达式2

      执行流程：
          A:计算关系表达式的值，看是true还是false
          B:如果是true，表达式1就是结果
            如果是false，表达式2就是结果
